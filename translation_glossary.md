# Enigmatica 整合包 AI 翻译术语库

> 基于 Enigmatica Sky (ES) 整合包中文翻译润色过程中积累的术语规范。
> 适用于 FTB Quests、KubeJS、模组语言文件等翻译场景。

---

## 一、核心翻译原则

1. **以社区包为标准** — 优先参考 MC百科、Minecraft-Mod-Language-Modpack 社区翻译包
2. **同类术语保持统一** — 同一英文词在所有模组中采用相同译法
3. **保留原文梗/彩蛋** — 如 TECNOLOGIA、文学作品梗（海明威等）
4. **技术术语准确优先** — 机械零件、化工术语使用行业通用译名
5. **药水/箭矢格式对齐原版** — 遵循 Minecraft 原版命名规范

## 三、通用翻译规范

### 药水命名

遵循原版格式：
- 普通：`X药水`（如 `力量药水`）
- 喷溅型：`喷溅型X药水`
- 滞留型：`滞留型X药水`（注意不是"喷溅型"）
- 箭矢：`X之箭`（不使用"箭矢·X"格式）

| English | 中文 |
|---------|------|
| Greater Strength | 力量 |
| Greater Regeneration | 再生 |
| Greater Mana Regen | 魔力恢复（注意统一用"魔力"） |
| Greater Spell Damage | 法术伤害 |
| Greater Shielding | 护盾 |
| Greater Recovery | 恢复 |
| Flight / Long Flight | 飞行 |
| Resistance / Long / Strong / Greater | 抗性 |

### 村民交易信息格式

```
通过与[职业名]村民交易获得
交易等级：[等级]
```

### 盔甲/装备描述

使用"穿戴全套XX套装"格式（不用"穿着"）。

### 流体描述

- 静态流体：`流体名`
- 流动状态：`流动的流体名`
- 桶装：`流体名桶`

### 格式代码使用

| 代码 | 颜色 | 用途 |
|------|------|------|
| §a | 绿 | 正向提示、特殊物品名 |
| §5 | 深紫 | 歌词/彩蛋 |
| §7 | 灰 | 副标题、说明文字 |
| §c | 红 | 警告 |

## 四、常见翻译陷阱

| 英文 | 错误译法 | 正确译法 | 说明 |
|------|---------|---------|------|
| Source Berry | 覆盆子 | 魔源浆果 | raspberry 才是覆盆子 |
| Moa | 鸸鹋 | 恐鸟 | 鸸鹋是澳洲鸟类，恐鸟是灭绝巨鸟 |
| Sheepuff | 羊绒 | 云跳羊 | 不是"羊绒" |
| Glazed | 釉面 | 甜的/糖衣的 | 食物语境下指糖衣，非陶瓷釉面 |
| sus | 萨斯 | 可疑的 | suspicious 缩写，不是人名 |
| Geomancer | 地质学家 | 大地术士 | 维持 -mancer 后缀一致 |
| Andesite Alloy | 安山岩合金 | 安山合金 | MI 社区约定俗成 |
| Vibrant | 鲜活的 | 振谐 | MI 语境中为"振动谐振" |
| Alluvial Trommel | 淘金机 | 冲积式滚筒筛 | 技术术语 |
| Catching Fire | 着火 | 燃火 | Hunger Games 梗，保持文学性 |
| Sculk | 闪烁 | 幽匿 | Minecraft 官方中文 |
| Qulliq | （音译） | 石灯 | 因纽特文化特有的石灯 |
| Nyx | 尼克斯 | 夜之 | 希腊神话夜神 |
| The Bun Also Rises | 面包也升起 | 面包照常升起 | 海明威《太阳照常升起》梗 |
| Sun's Out Buns Out | 太阳出来包子出来 | 艳阳高照，包子出笼 | 俚语谐音梗 |
| Glazed and Confused | 釉面与困惑 | 甜到发懵 | 电影 Dazed and Confused 梗 |
